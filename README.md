## 📘 Contexte et Description du projet

Ce projet a été réalisé dans le cadre du cours **« Introduction à la programmation avec Python »**.  
Son objectif est de mettre en pratique les différentes notions abordées durant le cours, tout en développant une application permettant de proposer une **solution concrète pour l’analyse ou la gestion financière**.

À travers ce projet, plusieurs concepts de programmation et d’analyse de données sont mobilisés, notamment :

- la manipulation de données avec **Python** et **Pandas** ;
- la création de visualisations interactives avec **Plotly** ;
- le développement d’une **application web interactive** avec **Streamlit** ;
- l’intégration de fonctionnalités d’**intelligence artificielle** via l’API **OpenAI**.

## ✨ Fonctionnalités

- **Visualisations interactives** : graphiques dynamiques permettant d’explorer les tendances de dépenses par période.  
- **Analyses personnalisées** : génération de recommandations personnalisées basées sur l’analyse des données.  
- **Filtres personnalisables** : segmentation des données selon différents critères (type de dépense, périodicité, etc.).  
- **Tableaux de bord** : vue d’ensemble des principaux indicateurs clés.

## 🚀 Installation et utilisation

Il est recommandé d’utiliser **GitHub Codespaces**, ce qui permet d’exécuter le projet directement dans le navigateur sans installer Python ou les dépendances sur votre ordinateur.

1️⃣ Ouvrir le repository sur GitHub.

2️⃣ Cliquer sur **Code** puis sélectionner **Codespaces**.

3️⃣ Cliquer sur **Create codespace on main** pour lancer l’environnement de développement.

4️⃣ Une fois le Codespace ouvert, installer les dépendances dans le terminal :

```bash
pip install -r requirements.txt
```
